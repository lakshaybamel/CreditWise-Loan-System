# 💳 CreditWise Loan Approval System

## 📌 Overview
This project focuses on building a Machine Learning model to predict loan approval decisions based on applicant data.

The goal is to assist financial institutions in making faster, more accurate, and unbiased loan approval decisions.

---

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Train multiple classification models
- Evaluate models using Precision, Recall, and F1-score
- Identify the best-performing model

---

## 📊 Dataset Features
- Applicant Income
- Coapplicant Income
- Employment Status
- Age
- Marital Status
- Dependents
- Credit Score
- Existing Loans
- DTI Ratio
- Savings
- Collateral Value
- Loan Amount
- Loan Term
- Loan Purpose
- Property Area
- Education Level
- Gender
- Employer Category

---

## 🤖 Models Used
- K-Nearest Neighbors (KNN)
- Logistic Regression
- Naive Bayes

---

## 📈 Results
- Naive Bayes performed best based on evaluation metrics
- Important features: Credit Score, Income, DTI Ratio
